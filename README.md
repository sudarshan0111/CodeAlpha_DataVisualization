# Titanic Dataset - Data Visualization

## Project Overview

This project focuses on visualizing the Titanic Dataset using Python visualization libraries. Different chart types were used to uncover patterns, relationships, and trends in passenger data.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Visualizations Created

1. Passenger Survival Percentage (Pie Chart)
2. Gender Distribution (Bar Chart)
3. Average Fare by Passenger Class (Line Chart)
4. Fare Distribution by Passenger Class (Box Plot)
5. Correlation Heatmap
6. Age vs Fare (Scatter Plot)

## Key Insights

* Around 61.6% of passengers did not survive.
* Male passengers outnumbered female passengers.
* First-class passengers paid significantly higher fares.
* Fare values contain several outliers.
* Passenger class and fare have a notable relationship with survival.
* Age alone does not strongly determine survival.
